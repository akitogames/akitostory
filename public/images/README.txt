【イラストの配置方法】

このフォルダに以下のファイル名でPNG画像を配置してください。
画像がない場合は暗いグレーの背景で代替表示されます。

【必要な画像一覧】
scene1.png      → 深夜にスマホでガチャを引くアキト
scene2.png      → 荒れた部屋、生活が苦しくなったアキト
scene3.png      → 督促状の山、追い詰められたアキト
scene4.png      → 地下闘技場のリング
scene5.png      → 殴られて倒れるアキト（死亡シーン）
scene6.png      → 自室で目覚めるアキト
scene7.png      → スマホを見つめるアキト
scene_good.png  → アキト更生BOTを使い始めるアキト（希望のシーン）
scene_bad.png   → また課金するアキト（絶望のシーン）

【推奨サイズ】
横長（16:9） 例: 1280x720 または 1920x1080
